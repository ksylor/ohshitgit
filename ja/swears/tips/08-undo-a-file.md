```git
# ファイルが変更される前のコミットのハッシュを見つけよう
git log
# 矢印キーで履歴内を上下して、コミットを見つけたらハッシュを保存して
git checkout [saved hash] -- path/to/file
# インデックスには古いバージョンのファイルがあるはず
git commit -m "やった！ コピペして元に戻す必要ないじゃん"
```
